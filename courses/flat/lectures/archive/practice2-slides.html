NFA Practice

Design an NFA for each of the languages below. For each language, Σ = {a, b}.

Given two NFAs:

A = { w | w ends in ab } B = { w | w contains a substring aba} C = { w | w has an odd number of a’s }

  1. Construct an NFA that will recognize A ∪ B
  2. Construct an NFA that will recognize A ∪ C
  3. Construct an NFA that will recognize B ∪ C
  4. Construct an NFA that will recognize A ∘ B
  5. Construct an NFA that will recognize A ∘ C
  6. Construct an NFA that will recognize B ∘ C
  7. Construct an NFA that will recognize A*
  8. Construct an NFA that will recognize B*
  9. Construct an NFA that will recognize C*
